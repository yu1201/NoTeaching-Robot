## 主界面

主界面已经统一为深色风格，当前主要分成三块：

- 机器人监控
- 日志/坡口相机数据
- 常用功能入口

### 当前主界面入口

- 发送常驻服务
- 工艺参数
- 功能测试
- 先测后焊
- 精测量数据
- 点动控制
- FANUC 连接
- FANUC 断开
- 坡口相机测试
- 相机参数

## 机器人监控

监控区当前显示：

- 机器人运行状态
- 机器人累计毫秒
- PC 接收毫秒
- 当前直角坐标
- 当前关节脉冲

刷新逻辑已做“有变化再更新”，避免窗口持续无意义重绘。

## 子界面

### 功能测试

用于集中做 FANUC 基础功能验证：

- 设置速度
- 读取当前位置
- 读取关节脉冲
- 检查运行完成
- 写读 INT 寄存器
- 调用任务
- 发送 LS 程序
- CURPOS 诊断
- MOVL 往返测试
- MOVJ J2/J3 测试
- 零位运动

### 点动控制

点动界面已经支持：

- 单击固定步长点动
- 长按连续点动
- 读取当前位置到编辑框
- 手动修改目标值
- 运动到指定位置
- 分别设置直角速度、关节速度
- 速度参数持久化保存

### 工艺参数

工艺参数界面当前结构：

- 左侧工艺列表
- 左侧焊道列表
- 右侧 `常规工艺参数` / `摆动参数` 双标签页

支持：

- 新增/删除工艺
- 新增/删除焊道
- 修改参数
- 保存到本地配置文件

### 精测量数据修改

当前支持：

- 从控制单元中读取机器人列表
- 切换机器人驱动
- 示教和保存下枪安全位、扫描起点、扫描终点、收枪安全位
- 按 ini 注释分组显示其它参数
- `重新读取` / `保存参数`

### 相机参数

当前支持：

- 选择机器人
- 选择相机
- 读取/修改相机基础参数
- 打开手眼矩阵参数窗口

### 手眼矩阵参数

当前支持：

- 每个机器人下再区分相机
- 修改旋转矩阵和位移向量
- 独立保存手眼矩阵 ini

### 先测后焊

当前界面入口为：

- 预设参数
- 线扫处理

其中 `预设参数` 已接主流程，`线扫处理` 仍为空函数。

## CLI 调试入口

为了便于本机联调，程序当前还支持若干命令行入口：

- `--open-function-test`
- `--open-jog`
- `--open-precise-measure`
- `--open-camera-param`
- `--fanuc-connect`
- `--fanuc-upload-services`

这些入口适合在本机快速验证子界面和 FANUC 服务逻辑。
